<h1 align="center">Setup audio on Chromebooks</h1>

# Instructions
1. `git clone https://github.com/eupnea-linux/audio-scripts`
2. `cd audio-scripts`
3. `./setup-audio`

# Requirements
1. `python 3.10`

# Supported distros
1. Ubuntu 22.10
2. Debian testing
3. Fedora 37
4. OpenSUSE
5. Void Linux
6. Arch Linux

Other distros will likely work but will require you to manually install packages.
